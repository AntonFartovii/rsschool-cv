# Вёрстка для Desktop без адаптива

Вёрстка выполнена основными блоками  
header 
main 
footer 

В каждом блоке контейнер для выранивание контента во всех блоках
```
.container {
    max-width: 1440px;
    margin: 0 auto;
}
```
4 sections: preview, steps, destinations, stories

в section  preview, steps, stories inner-wrapper регулирует границы контаента внутри 
```
.inner-wrapper {
    position: relative;
    max-width: 1170px;
    margin: 0 auto;
}
```
